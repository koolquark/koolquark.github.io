Python Code : Interleave two files line by line.

Posted on Mon 13 November 2017 in Python

Interleave two files line by line

The following python code displays the interleaved output from two files on a line by line basis. Also the characters differing are colored different. The code is very basic. colorama python package is used to generate colored output.

import sys
from colorama import init

from colorama import Fore, Back, Style
init()

filename1 = sys.argv[1]
filename2 = sys.argv[2]

print("Filenames : {} and {}".format(filename1,filename2))

with open(filename1,'r') as f1, open(filename2, 'r') as f2:
    for la,lb in zip(f1,f2):
        acla = ""
        aclb = ""
        for c1,c2 in zip(la,lb):
            if c1 == c2:
                acla += Fore.GREEN + c1
                aclb += Fore.BLUE + c2
            else:
                acla += Fore.RED + c1
                aclb += Fore.RED + c2
        print(acla)
        print(Style.RESET_ALL)
        print(aclb)
        print(Style.RESET_ALL)
